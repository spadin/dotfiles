# Dotfiles

This repository is mainly here to aid me in quickly setting up my command line environment on new machines or fresh installs.

Feel free to use any config settings within this repository.

## Pre-requisite

This repository is a castle for [Homeshick](https://github.com/andsens/homeshick).

## Installing Homeshick

    $ git clone git://github.com/andsens/homeshick.git $HOME/.homesick/repos/homeshick

Then source the following from your `.zshrc`

    source "$HOME/.homesick/repos/homeshick/homeshick.sh"

## Cloning and using this castle

    $ homeshick clone spadin/dotfiles
    $ source ~/.zshrc
